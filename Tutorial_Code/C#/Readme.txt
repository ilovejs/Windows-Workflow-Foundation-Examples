The Solution files in the root folder contain the completed project for Steps 1-3 of
the WF45 Getting Started Tutorial downloaded from http://aka.ms/WF45GettingStarted

The subfolders contain the completed projects for each step of the tutorial.

Step1 - How to Create an Activity
Step2 - How to Create a Workflow
Step3 - How to Run a Workflow
Step4 - How to Create and Run a Long Running Workflow
Step5 - How to Create a Custom Tracking Participant
Step6 - How to Host Multiple Versions of a Workflow Side-by-Side
Step7 - How to Update the Definition of a Running Workflow Instance

The Form subfolder contains the Windows Form that is added in Step4.
